💬 과제 소감

Task 4를 마치며

CSS 변수(Custom Properties)를 활용한 디자인 토큰 시스템의 위력을 직접 체험했습니다. --md-primary 같은 변수 하나를 바꾸면 페이지 전체의 색상이 자동으로 변경되는 경험은 실제 프로젝트에서도 꼭 활용하고 싶은 패턴이라고 느꼈습니다.

M3 디자인 시스템의 타이포그래피 스케일, Shape 토큰, Elevation 시스템을 적용하면서 일관성 있는 UI가 왜 중요한지 이해했습니다. 색상, 글꼴, 여백이 규칙에 따라 조화를 이루면 사용자가 페이지를 더 신뢰하고 편하게 느낀다는 것을 실감했습니다.
