핵심 CSS Grid 코드

/* 그리드 컨테이너 */
.photo-grid {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  grid-template-rows:    repeat(3, 1fr);
  gap: 10px; /* 사진 간 간격 */
}

.photo-card {
  aspect-ratio: 1 / 1;     /* 정사각형 유지 */
  overflow: hidden;
  border-radius: 12px;   /* M3 Medium Shape */
}

.photo-card img {
  width: 100%; height: 100%;
  object-fit: cover;      /* 비율 유지하며 채움 */
}

💬 과제 소감

Task 2를 마치며

CSS Grid의 repeat(3, 1fr) 단 한 줄로 완벽한 3×3 격자를 만들 수 있다는 것이 놀라웠습니다. 이전에는 float이나 inline-block으로 격자를 구현하면서 복잡한 계산이 필요했는데, Grid를 사용하면 훨씬 직관적입니다.

aspect-ratio: 1 / 1 속성으로 이미지를 항상 정사각형으로 유지하는 방법도 새롭게 배웠습니다. hover 시 scrim(반투명 오버레이)으로 사진 설명을 보여주는 인터랙션을 추가하면서 시각적 완성도도 높일 수 있었습니다.
