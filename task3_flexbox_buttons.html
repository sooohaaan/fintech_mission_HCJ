Flexbox로 버튼 정렬

요구사항 Flexbox를 사용해 중앙에 정렬된 두 개의 버튼을 디자인합니다. 하나는 "확인", 다른 하나는 "취소" 버튼으로, 버튼에 배경색과 호버 효과를 적용해 봅니다. 결과물로 버튼 두개가 있는 웹 페이지의 스크린샷을 제출합니다. Flexbox로 버튼 중앙 정렬 "확인"과 "취소" 버튼 디자인 버튼에 배경색과 호버 효과 적용

핵심 Flexbox 코드

/* 버튼 컨테이너 */
.btn-group {
  display: flex;
  justify-content: center;  /* 가로 중앙 */
  align-items: center;
  gap: 12px;
}

/* M3 Filled Button */
.btn-filled {
  height: 40px; padding: 0 24px;
  border-radius: 50px; /* Full shape */
  background: var(--md-primary);
  color: var(--md-on-primary);
}

/* M3 Outlined Button */
.btn-outlined {
  border: 1px solid var(--md-outline);
  background: transparent;
  color: var(--md-primary);
}

💬 과제 소감

Task 3을 마치며

Flexbox의 justify-content: center 한 줄로 버튼을 완벽하게 가로 중앙 정렬할 수 있다는 것이 인상적이었습니다. 예전에는 margin: auto나 position: absolute로 복잡하게 처리했는데, Flexbox를 통해 훨씬 직관적이고 유지보수하기 좋은 방식을 익혔습니다.

확인/취소 버튼에 각각 Filled와 Outlined 스타일을 적용하면서 M3 버튼의 위계 구조도 이해할 수 있었습니다. Modal Dialog를 통해 사용자의 행동에 즉각적인 피드백을 주는 UX의 중요성도 배웠습니다.
