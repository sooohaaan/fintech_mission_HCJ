💬 과제 소감

Task 5를 마치며

HTML의 시맨틱 태그들이 단순히 텍스트를 표시하는 도구가 아니라, 문서의 의미와 구조를 나타낸다는 것을 다시 한번 느꼈습니다. h1~h6의 계층, ul과 ol의 차이, blockquote로 인용문을 표현하는 것 모두 사용자와 검색엔진 모두를 위한 설계임을 이해했습니다.

개인 소개라는 친숙한 주제로 HTML 기본 태그를 연습하면서 딱딱하게만 느껴지던 마크업이 훨씬 자연스럽게 와닿았습니다. 특히 웹 접근성 관점에서 올바른 태그 선택이 얼마나 중요한지 고민하는 계기가 되었습니다.
